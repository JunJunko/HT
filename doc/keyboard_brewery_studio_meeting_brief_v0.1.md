# 《击酒师》工作室会议简版 V0.1


> 目标：确认产品核心、下一版 Demo 范围和暂不开发的内容

---

## 1. 我们想做什么

《击酒师》是一个常驻桌面角落的单机陪伴游戏。

玩家工作、聊天或写代码时产生的自然键盘输入，会在后台慢慢酿成不同的酒。大约每四分钟，一枚气泡升起并揭晓结果；不同的酒会吸引不同客人，并解锁原料、配方线索和控制工具。

我们希望玩家产生的感觉是：

> “我刚才只是正常做自己的事，结果这些输入居然酿出了一杯很像我的酒，而且真的有人喜欢它。”

这个项目的辨识度不是“按键增加生产进度”，而是：

```text
现实输入形成一杯酒
→ 气泡把出酒变成惊喜
→ 客人让这杯酒被理解
→ 玩家逐渐学会复现和控制
```

---

## 2. 一次理想的游玩体验

玩家打开游戏，将小酒馆放在屏幕角落，然后继续正常使用电脑。

### 第一杯：知道游戏不会打扰自己

四分钟后，一个普通气泡安静升起。玩家点击后得到“酒馆基础拉格”。游戏解释：这段时间输入较少，酿造结果接近酒馆的基础配方。

玩家不点击也没有损失，结果会自动收进酒架，之后可以补看。

### 第二杯：发现输入确实有影响

玩家集中打了一段文字。第二个气泡出现颜色和粒子，揭晓“黄昏淡艾尔”。游戏说明：发酵阶段的节奏比玩家平时更快，所以形成了暖发酵和果香倾向。

玩家开始相信：“这杯酒可能真是我刚才酿出来的。”

### 第三杯：酒和客人产生关系

一位喜欢暖发酵和果香的客人来到酒馆。他喝完后明显开心，并送给玩家一种新原料。玩家可以把原料装备到下一杯中，开始对结果进行轻量控制。

### 后续：出现值得分享的意外

某次气泡突然倒流或消失，揭晓一杯隐藏酒。玩家只获得部分配方线索，可以保存酒卡，或把挑战码发给朋友，让别人尝试复现。

完整体验是：

```text
自然输入 → 意外出酒 → 理解原因 → 客人回应
→ 获得原料或工具 → 尝试控制 → 发现稀有结果
```

## 3. 六条核心设计原则

### 1. 输入创造差异，但不代表能力高低

打得更多、打得更快，不等于品质更高。

输入活跃会带来更多配方方向；安静工作或暂时离开电脑，也会正常酿出基础酒。基础酒如果非常符合配方，同样可以是完美品质。

这样玩家不需要为了游戏刷键，也不会把工作效率变成游戏成绩。

### 2. 出酒有抽卡感，但不是纯随机抽卡

每次输入会形成一组“酿造指纹”，系统根据这组指纹寻找最符合的酒。

- 普通酒条件宽，容易自然出现；
- 高级酒需要几个输入特征互相配合；
- 史诗酒需要更准确的组合和特定原料；
- 隐藏酒还需要时间、客人、连续酒款或剧情条件。

随机只用于两个结果非常接近的时候。相似的输入应该经常得到相似的酒类。

### 3. 稀有度、品质和首次发现是三件事

| 维度 | 示例 |
|---|---|
| 配方稀有度 | 普通、高级、史诗、隐藏 |
| 本杯品质 | 粗糙、标准、优秀、完美 |
| 发现状态 | 首次发现、已收录 |

所以普通酒可以是完美品质，史诗酒也可能只是标准品质。普通酒不等于废卡。

### 4. 客人负责回应，不负责给玩家打低分

取消玩家可见的 0～100 满意度。

游戏仍会在后台判断哪位客人最喜欢这杯酒，但玩家看到的是角色反应：

- 合口：符合大类偏好；
- 很喜欢：多个风味吻合；
- 命中灵魂：触发礼物、剧情或配方线索。

如果没有具名客人喜欢，这杯酒会进入酒架或被普通路人消费。不会出现嫌弃、关系倒退、金币惩罚，也不会强迫客人喝不喜欢的酒。

### 5. 重复酿造增加掌握程度，不解锁完全自动复制

重复酒仍然有价值：

| 次数 | 解锁内容 |
|---:|---|
| 第 1 次 | 酒名和基础风味 |
| 第 3 次 | 主要形成原因 |
| 第 5 次 | 模糊配方范围 |
| 第 10 次 | 配方预设：装备推荐原料、显示偏差、锁定一个变量 |
| 首次完美 | 大师标记或专属外观 |

配方预设帮助玩家控制结果，但现实输入仍要参与。否则游戏最核心的“我的输入酿出了这杯酒”会消失。

### 6. 游戏必须尊重隐私和注意力

- 不保存实际输入字符；
- 不记录软件、窗口、文件名或聊天内容；
- 只保留字母、数字、符号、节奏和停顿等汇总信息；
- 随时显示“正在统计 / 已暂停”；
- 可以一键暂停；
- 气泡不抢焦点；
- 不点击气泡也会正常结算；
- 所有揭晓都能跳过、补看和静音。

---

## 4. 为什么要重点做气泡

Demo 中的气泡已经表现出成为项目标志的潜力。建议把它从普通完成通知升级为“出酒揭晓器”。

| 稀有度 | 气泡表现 |
|---|---|
| 普通 | 透明、小型、快速揭晓 |
| 高级 | 颜色渐变、微粒、先出现风味标签 |
| 史诗 | 膨胀和停顿，环境与客人产生明显反应 |
| 隐藏 | 倒流、消失、黑雾等异常现象 |

隐藏结果不应该只是“比史诗更金”。玩家第一眼应该觉得：“这次不对劲。”

抽卡感来自演出和未知结果，但它不能强迫玩家每四分钟切回游戏。

## 5. 游戏怎样自然产生传播

我们最值得优先验证的是：

> **个性酒卡 + 气泡揭晓回放 + 酿酒挑战码**

### 个性酒卡

显示酒名、稀有度、品质、三个成因标签、客人反应、游戏名称和挑战码。

玩家分享的重点不是“我抽到了一张金卡”，而是：

> “我正常工作时居然酿出了这个，你能复现吗？”

### 气泡回放

游戏内部重播约六秒的揭晓过程，只渲染游戏画面，不录制玩家桌面。

### 挑战码

朋友输入挑战码后获得目标和部分线索，但不会直接得到这杯酒。他可能成功复现，也可能酿出变体或发现隐藏分支。

分享因此不只是展示结果，而是邀请别人参与下一次探索。

## 6. 世界酒谱应该是什么

我们希望全部玩家都能看到特殊发现，但不建议制作传统总分排行榜。

“总按键数”“挂机时间最长”“打字速度最快”只会鼓励刷键，也会让新玩家永远追不上老玩家。

建议把在线系统命名为“世界酒谱”，首版只做三类内容：

1. **首次发现**：某隐藏配方的首位、前十位和里程碑发现者；
2. **本周奇闻**：罕见、有趣、离谱的酿酒事件；
3. **社区挑战**：参与人数或产生变体最多的挑战。

玩家点开一条记录后，应该能查看酒卡、部分线索和气泡回放，并直接发起挑战。榜单必须是探索入口，而不只是展览。

联网必须由玩家主动选择。拒绝联网也能获得全部单机配方、客人和工具；在线荣誉只奖励称号、外观和纪念内容。

建议在传播 Demo 证明玩家真的愿意交换挑战之后，再开发世界酒谱服务器。

## 7. 下一版 Demo 建议范围

下一版的目标不是增加大量内容，而是证明玩家是否愿意等待、理解、保存和交换一次出酒结果。

### 建议制作

- 8～10 种酒；
- 4～6 位客人；
- 至少 1 种史诗酒和 1 种隐藏酒；
- 普通、高级、史诗、隐藏四套气泡表现；
- 1 条完整链路：新酒 → 新客人 → 新原料 → 尝试复现；
- 酒卡；
- 游戏内气泡回放；
- 本地挑战码；
- 一套配方掌握进度；
- 保存酒卡、复制挑战码、导入挑战等测试记录。

### 暂时不做

- 世界酒谱服务器；
- 十连抽和卡池轮换；
- 角色升星和碎片；
- 赛季通行证；
- 高压力限时活动；
- 复杂库存、员工、房租和供应链；
- 完整装修系统；
- 实时多人；
- 20～30 种完整酒和大量角色美术。

## 8. 下一版 Demo 要回答什么

1. 玩家是否相信自己的输入影响了酒？
2. 玩家能否说出这杯酒形成的一个主要原因？
3. 普通、高级、史诗和隐藏是否一眼可区分？
4. 气泡是否有趣，但不打扰正常工作？
5. 客人反应是否比单纯获得金币更吸引人？
6. 普通酒是否仍然让玩家觉得有价值？
7. 玩家是否真的保存了酒卡？
8. 玩家是否真的把挑战码发给了别人？
9. 收到挑战的人是否愿意尝试？
10. 挑战失败产生的变体是否仍然有趣？

| 议题 | 当前建议 | 会议结论 |
|---|---|---|
| 产品核心 | 现实输入 → 气泡揭晓 → 客人回应 → 获得控制力 |  |
| 气泡地位 | 作为项目核心视觉和奖励时刻 |  |
| 稀有结构 | 普通 / 高级 / 史诗 / 隐藏 |  |
| 客人反馈 | 取消满意度分数，使用正向反应层级 |  |
| 重复酿造 | 第 10 次解锁配方预设，不自动复制 |  |
| 传播 Demo | 优先完成酒卡、回放和本地挑战码 |  |
| 在线功能 | 世界酒谱延后到传播行为被验证之后 |  |
| 内容范围 | 8～10 种酒、4～6 位客人 |  |
| 角色风格 | 人类 / 动物拟人 / 混合，待讨论 |  |
| 文案语气 | 治愈 / 幽默 / 轻微职场吐槽，待讨论 |  |
| 隐藏演出 | 奇幻 / 都市夜色 / 轻微怪诞，待讨论 |  |
| 首条解锁链 | 选择一杯酒、一位客人和一种原料 |  |

会议最后只需要得到三项明确结果：

1. 我们共同认可的一句话产品核心；
2. 下一版 Demo 的固定范围；
3. 第一条完整解锁链的酒、客人与原料。

## 10. 当前推荐结论

如果没有更强的反对理由，建议按以下方向推进：

> 保留桌面角落和现实输入，把气泡揭晓升级为核心体验；稀有度由配方条件决定，品质与稀有度分开；客人只提供正向回应和探索动力；重复酿造增加控制能力，但不取代现实输入；下一版先验证酒卡、回放和挑战码，之后再决定是否投入世界酒谱服务器。

我们最终希望玩家记住的是：

> “我正常生活和工作时，无意中酿出了一杯有性格的酒；气泡把这一刻变成惊喜，客人让这杯酒被理解，而朋友还可以继续挑战它。”
