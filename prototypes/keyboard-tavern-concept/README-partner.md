# HitTheBeer · 敲酒师（macOS 概念原型）

桌面陪伴小游戏概念原型：常驻屏幕角落的小窗口，你工作/写代码/聊天时的键盘输入会在后台慢慢酿成一杯啤酒，吸引不同客人来喝。本次是核心玩法测算。

---

## 第一次运行（重要，按顺序 3 步）

### 1. 解压
解压 `HitTheBeer-mac.zip`，得到 `HitTheBeer.app`。

### 2. 绕过 macOS 拦截（未签名 app）
**右键点击 `HitTheBeer.app` →「打开」**，弹窗里再选「打开」。
如果右键没有「打开」选项，打开「终端」运行一次（把路径换成实际的）：
```
xattr -dr com.apple.quarantine /存放路径/HitTheBeer.app
```

### 3. 授权键盘监听（必须！否则程序看起来不工作）
打开「**系统设置 → 隐私与安全性 → 辅助功能**」，在列表里找到 **HitTheBeer**，打开开关。
然后**退出 app 重新打开**一次，让权限生效。

> ⚠️ 这个游戏需要统计你在任何 app 里的键盘输入（否则没法酿酒），所以系统会要求「辅助功能」权限。
> 但它**只统计按键的类别和节奏**（字母/数字/符号、快慢停顿），**绝不记录你打的文字内容**，也**不联网**。可放心用于本次测试。

---

## 怎么玩

1. 开着它，去干你本来要干的活（写代码、聊天、写文档、回邮件）。
2. 角落小窗口里，酒液会随你的输入慢慢成形，**约 4 分钟一杯**。
3. 一杯完成时会弹出说明：这杯为什么是这个味道、哪位客人喝了、什么反应。
4. 点窗口标题栏的「**👤大厅**」按钮，展开成宽屏，能看到客人在酒馆里走动、酒保递酒、客人品尝的动画。
5. 点客人能看他的资料（偏好/喝过/关系）。

试着用不同方式工作（连续敲代码 vs 慢慢写文章 vs 录数字），看酿出的酒会不会不一样。

---

## 常见问题

- **酒液一直不动 / 不出酒**：99% 是没给「辅助功能」权限，或给权限后没重启 app。回到第 3 步。
- **双击打不开**：用「右键 → 打开」。
- **杀毒软件或系统提示「键盘记录」**：因为要统计全局键盘输入，属正常提示。本程序不记录文字、不联网。
- **闪退 / 打不开**：在终端里运行 `/存放路径/HitTheBeer.app/Contents/MacOS/HitTheBeer`，把报错截图发回来。
